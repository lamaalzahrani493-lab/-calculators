# 🧮 Advanced Scientific & Matrix Calculator

تطبيق ويب متقدم تم تطويره ليكون أداة شاملة للحسابات الرياضية العلمية ومعالجة المصفوفات (Matrix Operations) في واجهة واحدة عصرية.


## 💡 نظرة عامة
تم تصميم هذه الحاسبة لتكون أكثر من مجرد آلة حاسبة عادية؛ فهي توفر للمستخدمين (طلاباً ومبرمجين) إمكانية إجراء عمليات رياضية معقدة، وتدعم المصفوفات بأنظمة تبويب متطورة، مع واجهة مستخدم تعتمد على مبادئ الـ Glassmorphism.

## ✨ المميزات الرئيسية
* **نظام التبويبات (Tabs):** تنقل سلس بين الآلة الحاسبة، العمليات العلمية، والمصفوفات.
* **محرك مصفوفات متقدم:** إمكانية إدخال مصفوفات بأبعاد مختلفة (حتى 10×10) وإجراء عمليات (المحدد، المدور، العكس، الرتبة).
* **سجل العمليات (History):** حفظ العمليات الحسابية محلياً في المتصفح للرجوع إليها لاحقاً.
* **تصميم متجاوب (Responsive):** يعمل بكفاءة على كافة الأجهزة (الهواتف، الأجهزة اللوحية، والحواسيب).
* **دعم الأنظمة العددية:** تحويل الأرقام بين العشري، الثنائي، السداسي عشر، والثماني.

## 🛠 التقنيات المستخدمة
- **HTML5**: لهيكلة المحتوى.
- **CSS3**: تصميم عصري باستخدام Flexbox, Grid, و Glassmorphism.
- **JavaScript (Vanilla)**: المنطق البرمجي، إدارة الحالة، ومعالجة المصفوفات رياضياً.
